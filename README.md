Personal portfolio built with a focus on premium aesthetics, clean typography, and real-time data integration. Designed to showcase full-stack development expertise and experimental AI projects.

## Key Features

- **Premium UI/UX**: Minimalist, high-contrast design with glassmorphism effects and smooth scroll animations.
- **Real-time Git Logs**: Dynamic integration with GitHub API to display the latest commits and starred repositories.
- **Project Showcase**: Detailed overview of key projects including AI-powered platforms and modular Discord systems.
- **Tech Stack Visualization**: Clean categorization of tools and technologies.
- **Analytics Driven**: Integrated Vercel Analytics for tracking global visitor metrics.
- **Responsive & Performant**: Fully optimized for all device sizes with near-perfect Lighthouse scores.

## Built With

- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Custom Design System)
- **APIs**: GitHub REST API
- **Deployment**: Vercel

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/caya8205-2/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
